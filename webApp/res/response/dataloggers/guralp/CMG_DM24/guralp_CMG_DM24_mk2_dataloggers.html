CMG-DM24 Mk2 Datalogger

DM24 Mk2 Output Options

The Guralp CMG-DM24 Mk2 dataloggers have been manufactured with two output options having different bit weights:

  • single-ended ( bit weight 1.27 microVolts/count), and
  • differential ( bit weight 3.2-3.5 microVolts/count)
The responses given here use a nominal value of 3.2 microVolts/count (or 3.125e+05 counts/V) for the Mk2 with differential output.

As with all Guralp instruments, the manufacturer's calibration sheet for each serial number will list that instrument's most accurate bit weight. When updating a datalogger sensitivity using a nominal response, not only is it necessary to change the stage 3 (A/D stage) sensitivity value, but the stage 0 sensitivity representing the product of all stage gains must also be recalculated.
