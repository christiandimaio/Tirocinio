Guralp CMG-DM24 Dataloggers

About CMG-DM24 Dataloggers

Guralp CMG-DM24 dataloggers have been manufactured using two different A/D chips which have different nominal sensitivities and use different FIR filters for decimation:

  • Mk1, which uses a Crystal CS5322 chip and has a firmware version beginning with v.0 but is less than V.090
  • Mk2, which uses a Crystal CS5322 chip and was manufactured before 2004.
  • Mk3, which uses a Crystal CS5376 chip and was manufactured from 2004 on.
