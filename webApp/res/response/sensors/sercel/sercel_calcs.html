Calculating Response Information for Sercel Sensors

The equations needed to calculate sensitivity, damping and poles and zeros for Sercel L-4 and L-22 sensors depend on these equations where the coil resistance Rc and shunt resistance Rs are known:

  • G = sensitivity in V/M/S = A * sqrt(Rc)
  • bc = current damping = (B * Rc) / (Rc + Rs)
  • bt = total damping = bo + bc
  • p1, p2 = (-2 * pi * f0 * bt) +/- j * ( 2 * pi * f0 * sqrt(1 - bt**2) )
  • z1 = z2 = 0

where

  • bo = open circuit damping
  • f0 = fundamental frequency
  • p1,p2 = poles
  • z1,z2 = zeros

Other values for the L-4 and L-22 sensors are as follows:

Sensor f0 bo A B
L-4A 2 Hz 0.28 3.73 1.1
L-4C 1 Hz 0.28 3.73 1.1
L-22D 2 Hz 0.46 1.61 1.22

Note:

These calculations assume that the input resistance of the datalogger or amplifier attached to the sensor is at least 2 orders of magnitude greater than the shunt resisitance. If this is not the case, the total damping (and resulting poles) will also depend on the input resistance.
