Calculating Response Information for OYO Geospace Sensors Given Datalogger Input Impedance

A datalogger or preamplifier attached to a geophone acts as an additional resistance across the signal coil. This changes the damping, which in turn changes the pole values. The larger the impedance contrast between the sensor and datalogger, the smaller the effect. The larger the preamp gain setting, the larger the impedance tends to be. Here is a partial list of some common datalogger impedances.

The effect of the datalogger impedance on damping is

  • bt = total damping = bo + (damping constant / (Rc + Zd))
where (the numbers below are specific to an HS-1-LT MINI with a 2500 ohm coil):
  • bo = open circuit damping = 0.61
  • Rc = coil resistance = 2500 ohms
  • Zd = datalogger input impedance
  • damping constant = 6694 ohms
  • p1, p2 = (-2 * pi * f0 * bt) +/- j * ( 2 * pi * f0 * sqrt(1 - bt**2) )
The poles are then
  • p1, p2 = (-2 * pi * f0 * bt) +/- j * ( 2 * pi * f0 * sqrt(1 - bt**2) )

where

  • bo = open circuit damping
  • f0 = fundamental frequency
  • p1,p2 = poles
  • z1,z2 = zeros
