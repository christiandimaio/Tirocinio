Titan Sensitivities

The Titan accelerometer is available is several models, all of which are represented by the same responses. The Titan response is determined by the maximum full scale measurement range the user selects, whether +/-4g, +/-2g, +/-1g, +/0.5g +/-0.25g or +/-0.125g.

The Titan responses also apply to the accelerometer portion of the following integrated products, which contain a Titan accelerometer:

  • Trillium Cascadia Posthole, which contains a Titan accelerometer and a Trillium Compact seismometer.
  • TitanSMA, which contains a Titan accelerometer and a Centaur-type digital recorder
  • TitanEA, which contains a Titan accelerometer and a Centaur-type digital recorder

Nominal sensitivity in V/(m/s**2) is calculated by (Full scale voltage / Clip level) / 9.80665 m/s**2
